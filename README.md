PureLine Dresses – Minimalist Fashion Store
Tagline: Where Simplicity Meets Style

About the Project
PureLine Dresses is a modern online fashion store showcasing minimalist dress designs with a focus on clean lines, soft colors, and elegant simplicity. This full-stack project demonstrates scalable e-commerce architecture using Python (Flask), C# (.NET Web API), and a responsive web frontend.

Features
Minimalist fashion storefront
Product catalog with detailed dress descriptions
REST API backend
Responsive frontend layout
Product database support
Ready for payment gateway integration
Expandable for admin dashboards and user accounts
